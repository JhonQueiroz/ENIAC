
Como executar o algoritmo genético via terminal Linux/macOS:

Passo 1: Dê permissão de execução:
No terminal digite: 

chmod +x run_GA.sh

chmod +x run_GLF.sh

chmod +x run_GR.sh

Passo 2: Execute:

./run_GA.sh

./run_GLF.sh

./run_GR.sh

-----------------------------------------------------------------------------

Coloquei também um script que faz mais de uma execução em paralelo:
Execute os seguintes comandos no terminal do MAcOS:

chmod +x run_GA_parallel.sh
./run_GA_parallel.sh 4              # paraleliza com 4 processos ao mesmo tempo
